# POWER STREET RECORDS

A Recording Studio MLO for GTAV FiveM

Based on an existing cutscene Record A IPL/MLO.
Modified and fixed up multiple files/collisions to make it a more complete and finished 'MLO' by fixing some room bounds, removing some dead end rooms/doors to nowhere and various other bits and pieces of R* scuff basically. Positioned this MLO in the Lower Power St location near Union Depository with doors to walk straight into the mlo. The exterior banner out the front and logo inside reception are customisable within the included YTD file. Also includes custom interior audio occlusions.

### Location
Lower Power street, Pillbox Hill. </br>
Co-ords: 102.6092, -637.5572, 31.6746

### Installation
Download package from releases, unzip and drop the folder into you servers resources folder and then add it to your server.cfg to start the resource.
